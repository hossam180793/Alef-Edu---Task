Steps to run the code :

1- Download the project on your machine 
2- Open the project using eclipse editor 
3- Open Class "SpringTest.java" in "tests" Package
4- Right Click on the class "SpringTest.java" 
5- Select Run as >> TestNG Test

IMPORTANT NOTES	:-

- Please Match the ownerCount number to be matched with the number of records of the owners table
- Please increase the ownerCount Variable by 1 in SpringTestData.xlsx before each run.
